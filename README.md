# Smart Survey, Forms & Polling Platform (Vanilla JS)

A configurable frontend platform built with **HTML, CSS, and JavaScript**.

## Features

- Admin can create/edit/delete forms and polls dynamically
- User can fill dynamic forms and vote in polls
- Single vs Multiple submission enforcement for forms
- Single vs Multiple choice enforcement for polls
- Active/Inactive status toggles
- Dashboard with analytics
- Search and filter for forms/polls
- LocalStorage persistence

## Pages

- `index.html` - Landing page
- `admin.html` - Admin module (forms + polls management)
- `user.html` - User module (fill forms + vote polls)
- `dashboard.html` - Analytics dashboard

## Run

Just open `index.html` in browser.

## Project Structure

- `css/` page and component styles
- `js/constants/` app constants
- `js/services/` storage and business logic
- `js/modules/` admin/user/dashboard rendering modules
- `js/utils/` reusable helpers
- `js/state/` simple app store
